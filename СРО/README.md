# СРО Folder Structure

The СРО (Software Development Projects) folder contains various projects developed during the course. Each project has its own subfolder, organized as follows:

## Project Structure
- **ProjectName/**  
  - **src/**              - Contains the source code of the project.
  - **docs/**             - Documentation and reports related to the project.
  - **tests/**            - Unit tests and other testing resources.
  - **README.md**         - Overview and instructions for the project.
