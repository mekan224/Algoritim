# Project Report: Expression Calculator with Stack Implementation

### Project Overview
This project implements an expression calculator using stack data structures. The primary objective is to evaluate mathematical expressions provided in infix notation by converting them to postfix form using the Shunting Yard algorithm and then evaluating the resulting postfix expression.

### Key Features
1. **Stack-based Evaluation**: Utilizes stack data structures to evaluate expressions efficiently.
2. **Operator Precedence Handling**: Correctly handles operator precedence and associativity to produce accurate results.
3. **Error Handling**: Includes error checks for invalid expressions, ensuring robustness.

### Implementation Details
- **Infix to Postfix Conversion**: The implementation converts infix expressions to postfix (Reverse Polish Notation) using the Shunting Yard algorithm.
- **Postfix Evaluation**: Evaluates the postfix expression using a stack, ensuring that operations follow the correct order.

### Instructions
To use the calculator:
1. Input an infix expression such as `3 + 5 * ( 2 - 8 )`. 
2. The program will output the evaluated result of the expression.

### Conclusion
This project demonstrates the application of stack data structures in evaluating mathematical expressions, providing a solid foundation for further exploration of data structures and algorithms.
